➕ Tạo đề mới

Gõ câu hỏi vào form, chọn loại, đánh dấu đáp án đúng. Không cần code.

Quy trình mới: ① Điền thông tin đề → ② Thêm từng câu hỏi → ③ Bấm "Lưu đề & Xem lại" → ④ Kiểm tra đề trong bảng xem lại → ⑤ Bấm "Xác nhận & Đẩy lên GitHub". Xong! Web tự cập nhật cho mọi người.
🔑 Lần đầu đẩy, hệ thống sẽ hỏi GitHub Token (chỉ nhập 1 lần, lưu trên máy này). Cách tạo token xem trong file github-sync.js.
Công thức toán: gõ trong dấu $...$, ví dụ $x^2 + 1$. Xuống dòng cứ Enter bình thường.
