# 📚 Hệ Thống Ôn Tập Trực Tuyến

Hệ thống làm bài tập và kiểm tra trực tuyến cho học sinh THCS.

## ✨ Tính năng

- 📝 Làm bài trắc nghiệm trực tuyến
- ✍️ Làm bài tự luận (gõ hoặc đính kèm ảnh)
- ⏱️ Đếm ngược thời gian làm bài
- 🔒 Chế độ toàn màn hình, chống sao chép
- 💾 Tự động lưu kết quả vào Google Sheets
- 📊 Chấm điểm tự động cho phần trắc nghiệm
- 📱 Responsive - Hỗ trợ mobile

## 📂 Cấu trúc thư mục

```
Web bài tập/
├── index.html                          # Trang chủ
├── KHTN/                               # Đề thi KHTN
│   ├── de-so-1/
│   ├── de-so-2/
│   └── de-so-3/
├── Sinh-10/                            # Đề thi Sinh học 10
│   └── de-so-1/
├── template-de-trac-nghiem-tu-luan.html           # Template 1
├── template-de-full-trac-nghiem.html              # Template 2
├── template-de-trac-nghiem-dung-sai-tu-luan.html  # Template 3
└── HUONG-DAN-SU-DUNG-TEMPLATE.md      # Hướng dẫn
```

## 🚀 Cách sử dụng

### Cho học sinh:

1. Mở file `index.html` trên trình duyệt
2. Chọn môn học và đề thi
3. Nhập họ tên, lớp
4. Làm bài và nộp

### Cho giáo viên:

1. Sử dụng các file template để tạo đề mới
2. Xem hướng dẫn chi tiết trong `HUONG-DAN-SU-DUNG-TEMPLATE.md`
3. Kết quả học sinh được lưu tự động vào Google Sheets

- 🧬 Sinh học 10 (1 đề)
